# Frontend Mentor - Meet landing page

![My Meet Landing Screenshot](/src/assets/desktop/meetscreenshot.png)

A simple solution for the [Frontend Mentor](https://www.frontendmentor.io) challenge - Meet Landing Page.

This project was created to practive HTML & CSS & RWD following the disign in Figma.

---

## Overview

This project recreates a **Meet Landing Page** from a Figma design, including:

- Semantic HTML structure
- Responsive layout (mobile-first)
- Styled text hierarchy
- Optimized image display
- Clean and minimal CSS
